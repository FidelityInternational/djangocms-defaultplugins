====================
djangocms-defaultplugins
====================

Installation
============

Requirements
============

django CMS Default Plugins requires that you have a django CMS 4.0 (or higher)
project already running and set up.

To install
==========

Run::

    pip install djangocms-defaultplugins

Add ``djangocms_defaultplugins`` to your project's ``INSTALLED_APPS``.
